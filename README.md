# Chore Tracker with Supabase, React, and Vite

This application that tracks hours spent doing chores **React**, **Supabase**, and **Vite**. It allows users to track time spent on chore compared to roommates and partners.


## Features

- Track amount of time spent on householder chores
- Compare difference in roommates, partners, or friends doing house chore
- Analytics on the users time spent on chores

## Technologies Used

- **React**: Frontend library for building the user interface
- **Supabase**: Backend as a service for managing the database and authentication
- **Vite**: Build tool for fast development
- **PostgreSQL**: Database for storing todo items
- **ShadCN**: UI Component Library


## Setup

Follow these steps to set up the project locally.

### 1. Clone the repository

Clone this repository to your local machine:

```
git clone [insert http for repo]
```

### 2. Install dependencies

Install the required dependencies using npm:

```
npm install
```

### 3. Set up .env File

Create a .env file in the root directory with the following variables:
VITE_SUPABASE_KEY=
VITE_SUPABASE_URL=



### 4. Run the Application

Start the development server with:

```
npm run dev
```

Visit `http://localhost:5173` in your browser to see the Todo List app in action.
